## 输出格式

```
# 人物提示词

## [人物名]（ep01 新增）

### 即梦版（中文）
[完整叙事描述式提示词。单人竖屏全身像，简洁背景。
包含：风格声明、角色定位、面部细节、发型、体型、服装、配饰、鞋子、整体气质和站姿。
可直接复制到即梦/Seedream生成角色参考图。]

### MJ版（English）
[English narrative prompt, adapted for Midjourney syntax.
Same content, can append MJ parameters like --ar 9:16 --style raw]

---

## [人物名2]（ep01 新增）

### 即梦版（中文）
[...]

### MJ版（English）
[...]

---

## [人物名3]（ep02 新增）
（后续集数的新增人物追加在文件末尾）

---

## [人物名]（ep03 变体：冬装版）
（同一人物的变体追加在文件末尾，标注原版和差异）
```

## 输出规范

- 单人竖屏全身像，不使用三视图/设定图格式
- 每个人物必须同时输出中文版和英文版
- 中文提示词采用叙事描述式，不要用关键词堆叠式
- 英文版适配 Midjourney 语法
- 直接输出完整提示词，不要逐条解释设计理由
- 每个提示词必须完整可用，用户可以直接复制到文生图工具生成图片
- 每个条目标注创建集数（如 ep01 新增），便于追溯
- 服装/妆造描述必须与导演讲戏本中的服化道指导保持一致
- 复用项不重复写入，只为"新增"和"变体"项编写提示词
