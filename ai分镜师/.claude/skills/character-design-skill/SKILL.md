# 人物设定提示词技能

[技能说明]
    基于导演讲戏本中的「服化道指导」和剧本人物设定，为每个角色编写单人竖屏文生图提示词。
    每个角色输出中英双版，可直接用于即梦/MJ生成角色参考图。

    出图格式：单人竖屏全身像，简洁背景。

[信息来源]
    - 导演讲戏本（01-director-notes.md）→ 服化道指导（服装、妆造、道具）
    - 剧本人物设定 → 年龄、身高、体型、气质标签、穿衣风格
    - 讲戏本中的表演指导 → 角色气质、情绪基调

[执行流程]

    第一步：读取上游产物
        - 读取 01-director-notes.md 中的人物清单和各剧情点的服化道指导
        - 读取剧本中的人物设定部分
        - 读取 examples/character-prompt-examples.md
        - 读取已有 assets/character-prompts.md（如存在）
        - 只为"新增"或"变体"的人物设计提示词

    第二步：为每个人物编写双语提示词

        提示词必须包含（信息从服化道指导和剧本设定中提取）：
        - 角色整体定位（年龄、性别、族裔特征）
        - 面部细节：脸型、五官、肤色、表情
        - 发型/发色：具体描述发型样式
        - 体型：身高、体态
        - 服装：款式、颜色、材质、细节（来自服化道指导）
        - 配饰：手表、耳环、眼镜等（来自服化道指导）
        - 鞋子
        - 整体气质和站姿

        中文版：叙事描述式，for 即梦/Seedream
        英文版：对应翻译，适配 MJ 语法

    第三步：输出
        按 templates/character-prompts-template.md 格式写入 assets/character-prompts.md

    自检清单：
        □ 每个人物是否包含面部/体型/服装/配饰/气质全要素？
        □ 服装描述是否与讲戏本中的服化道指导一致？
        □ 是否同时有中文版和英文版？
        □ 风格是否匹配项目视觉风格？
        □ 人物之间是否有足够视觉区分度？
