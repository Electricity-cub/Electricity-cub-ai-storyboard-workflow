---

## 输出格式

```
# 场景提示词

## 场景01：[地点·角度/状态描述]（ep01 新增）

**镜头设置**：[景别] / [机位方向] / [光线状态] / [氛围]
**对应剧情点**：P01, P03（标注哪些剧情点会引用此场景）

### 即梦版（中文）
[完整叙事描述式提示词。
必须包含：视角/机位、景别、空间布局、前景/背景层次关系、
光源方向和色温、关键道具/物件、氛围情绪、风格声明。
不放人物，只描述纯环境。
可直接复制到即梦/Seedream生成场景参考图。]

### MJ版（English）
[English narrative prompt. Same content adapted for Midjourney.
Can append MJ parameters like --ar 9:16 --style raw]

---

## 场景02：[同一地点·不同角度/状态]（ep01 新增）

**镜头设置**：[景别] / [机位方向] / [光线状态] / [氛围]
**对应剧情点**：P05

### 即梦版（中文）
[...]

### MJ版（English）
[...]

---

（按镜头设置拆分，同一物理地点可能有多个场景条目）
```

## 输出规范

- ★ 场景按"镜头设置"划分，不按"物理地点"划分
- 同一地点不同机位/角度/光线 = 不同的场景条目
- 每个场景必须同时输出中文版和英文版
- 叙事描述式，不要关键词堆叠
- 每个条目标注创建集数和对应剧情点
- 场景参考图中不放人物，只描述纯环境
- 必须包含：视角/机位、前景/背景、光线方向和色温、关键道具
- 复用项不重复写入
