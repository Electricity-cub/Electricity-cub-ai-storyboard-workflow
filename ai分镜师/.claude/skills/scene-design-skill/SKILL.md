# 场景提示词设计技能

[技能说明]
    基于导演讲戏本中的「美术指导」「摄影指导」「灯光指导」「服化道指导」，为每个场景角度编写独立的文生图提示词。

    ★ 核心认知：场景参考图不是按"物理地点"划分，而是按"镜头设置"划分。
    同一个物理地点（如"甲板"），如果不同剧情点中有不同的：
    - 摄影机位/角度（如船尾看向船头 vs 低角度仰拍船头）
    - 前景/背景关系（如过肩看全景 vs 无前景的远景）
    - 光线状态（如灰蓝散射光 vs 暗金色光柱笼罩）
    - 美术搭建差异（如空甲板 vs 堆满物资箱的甲板）
    则需要生成不同的场景参考图。

    每个场景角度输出中英文两版：
    - 中文版：用于即梦（Seedream）/ 国内文生图工具
    - 英文版：用于 Midjourney / 海外文生图工具

[信息来源]
    - 导演讲戏本（01-director-notes.md）→ 美术指导（表演区、空间布局、场景搭建）
    - 导演讲戏本 → 摄影指导（机位、前景/背景、景深、构图角度）
    - 导演讲戏本 → 灯光指导（光源方向、色温、光比、氛围）
    - 导演讲戏本 → 服化道指导（场景中的关键道具）

[执行流程]

    第一步：从讲戏本提取场景角度
        - 读取 outputs/<集数>/01-director-notes.md
        - 逐个剧情点分析：美术指导中的表演区在哪？摄影指导中的机位放在哪？灯光指导中的光从哪来？
        - 为每个需要独立参考图的"机位+构图+光线"组合定义一个场景角度

        判断是否需要独立参考图的标准：
        - 同一地点、同一机位、同一光线 → 复用同一张参考图
        - 同一地点但机位/角度不同 → 需要新的参考图
        - 同一地点但光线状态显著不同（如日光→金色特效光）→ 需要新的参考图
        - 同一地点但美术搭建不同（如增加了大量物资箱）→ 需要新的参考图

    第二步：整理场景角度清单
        示例（以ep01甲板为例）：
        - 场景A：海面上空·极远景（空镜建立，灰蓝阴天）
        - 场景B：甲板·船尾向船头全景（灰蓝顶光，女生横七竖八躺着）
        - 场景C：甲板·中景女生群体（灰蓝光，恐慌的女生们坐起来）
        - 场景D：甲板·低角度仰拍船头（暗金色光柱笼罩凌羽）
        - 场景E：船舱门前·正面中近景（灰蓝侧光明暗分割）
        - ...

    第三步：为每个场景角度编写双语提示词
        提示词必须包含（从讲戏本对应维度提取）：
        - 视角/机位：从哪个位置看，什么景别（来自摄影指导）
        - 空间布局：表演区在哪，关键物件位置（来自美术指导）
        - 前景/背景：画面的层次关系（来自摄影指导）
        - 光线：主光方向、色温、明暗关系（来自灯光指导）
        - 关键道具/物件（来自服化道指导）
        - 氛围情绪
        - 风格声明

        中文版：叙事描述式，for 即梦/Seedream
        英文版：对应翻译，适配MJ语法，末尾加参数

        ★ 场景参考图中不放人物（只有环境），人物由人物参考图单独提供

    第四步：比对已有素材（ep02+时）
        - 读取已有 assets/scene-prompts.md
        - 已有且无变化的场景角度标注"复用"，不重复写
        - 新场景角度或状态变化的写新提示词追加

    第五步：输出
        按 templates/scene-prompts-template.md 格式写入 assets/scene-prompts.md

    自检清单：
        □ 每个视频提示词中引用的场景是否都有对应的参考图提示词？
        □ 同一地点不同机位/光线是否拆分为独立场景角度？
        □ 每个场景角度是否同时有中文版和英文版？
        □ 场景描述是否包含视角/机位、空间布局、前景/背景、光线、道具、氛围？
        □ 光线描述是否与讲戏本中的灯光指导一致？
        □ 场景参考图中是否不含人物（纯环境）？
