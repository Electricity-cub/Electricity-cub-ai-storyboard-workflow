# 分镜头脚本技能

[技能定位——和其他阶段的边界]

    导演讲戏 → 回答「为什么这样演」：情绪、潜台词、人物关系、留白意图
    ★ 分镜头脚本 → 回答「用什么镜头讲给观众看」：景别序列、插入镜头、运镜、节奏
    视频提示词 → 回答「Seedance执行什么」：镜头指令文本

    分镜阶段不重复讲戏本的情绪分析。
    分镜阶段的唯一输出是：精确的镜头表，每个S = 一条提示词的蓝图。

[★★★ 核心增值——插入镜头设计]

    分镜师的价值不是给剧本动作贴景别标签，
    而是增加剧本里没写但电影语法需要的镜头。

    剧本只写故事事件。分镜要把事件翻译成「镜头序列」，
    这个过程中必然需要增加剧本中不存在的镜头来完成视觉叙事。

    五种插入镜头类型：

    1. 细节插入（Detail Insert）
       功能：强调关键动作/道具的物理细节，让动作有"质感"
       触发：剧本中出现关键物理互动时
       例：
       - 剧本"一把抓住手腕" → 增加 ECU 五指收紧扣住手腕的特写
       - 剧本"木门关上" → 增加 ECU 门板与门框撞击的瞬间
       - 剧本"掌心白光消散" → 增加 ECU 光点在指缝间飘散
       - 剧本"咬着红唇" → 增加 ECU 牙齿咬住下唇的特写

    2. 反应插入（Reaction Insert）
       功能：动作-反动作原则（R07）——展示动作对旁人的影响
       触发：每个重要动作/台词之后
       ★ 反应镜头往往比动作镜头更重要——观众通过反应理解意义
       例：
       - 世界音宣布天赋后 → 增加女生们面面相觑、脸红、咬唇的反应快切
       - 洛凝雪说"我先来" → 增加凌羽微怔的表情反应
       - 金色光柱砸下来 → 增加女生们跌坐仰望的群体反应

    3. 环境插入（Cutaway）
       功能：建立空间/氛围/时间感，给叙事留"呼吸"
       触发：场景转换处、情绪段落之间
       例：
       - 海面浪涌的空镜（2-3秒，建立荒凉感）
       - 桅杆顶端残帆在风中抖动（转场过渡）
       - 甲板裂缝中渗入的海水（细节氛围）

    4. 主观镜头（POV Insert）
       功能：让观众进入角色视角，增强代入感
       触发：角色"看到"重要事物时
       例：
       - 凌羽POV：低头看到满甲板昏迷的女生
       - 洛凝雪POV：醒来第一眼看到无边大海
       - 白霜POV：低头看自己掌心的白光

    5. 转场插入（Transition Insert）
       功能：连接两个场景/情绪段，比硬切更丰富
       触发：相邻P之间情绪差距大时
       例：
       - P03→P04：天空骤然阴沉的ECU（从恐慌到超自然降临）
       - P05→P06：金色粒子飘落在女生肩头的ECU（从震撼到暧昧）

    ★ 插入镜头的标注规则：
    每个S必须标注类型：
    - [剧本] = 剧本中明确描写的内容
    - [细节INSERT] = 分镜师增加的细节插入
    - [反应INSERT] = 分镜师增加的反应插入
    - [环境INSERT] = 分镜师增加的环境插入
    - [POV INSERT] = 分镜师增加的主观镜头
    - [转场INSERT] = 分镜师增加的转场插入

[全集钩子设计（仅开头和结尾）]

    钩子只针对两个位置：
    1. 全集开头（前3-5秒）——留人，阻止划走
    2. 全集结尾（最后3-5秒）——续看，制造"下一集"欲望

    ★ 不是每个P都需要钩子。中间段落靠叙事节奏和镜头张力维持注意力。

    开头钩子设计原则：
    - 必须在3秒内制造视觉冲击或信息缺口
    - 竖屏短剧的开头钩子通常是：极致的美/性暗示/冲突/奇观
    - 第一帧就要有东西"抓眼睛"，静止开场=死亡

    结尾钩子设计原则：
    - 最后一个镜头必须留悬念或想象空间
    - 常用手法：关门后的声音暗示、未完成的动作、意味深长的表情定格
    - 让观众产生"接下来发生了什么"的欲望

[电影语法规则（精简版）]

    R01 180度规则：对话双方的关系线，所有机位在同一侧
    R03 动作方向一致：衔接镜头中运动方向保持一致
    R04 视线方向相反：对视双方视线方向必须相反
    R05 分截高度：腋下/胸下/腰下/臀下/膝下，避免关节处
    R07 动作-反动作：每个动作配反应镜头，反应比动作更重要
    R08 接近与远离：开场宽→推进紧→高潮特写→呼吸回宽
    R10 运动三段式：起幅静止→运动→落幅静止
    R13 30度规则：相邻镜头角度变化≥30度
    R14 景别跨级：相邻镜头至少跨2个景别级别

[每个S的输出规格]

    每个S直接对应一条Seedance提示词，因此必须包含：

    | 字段 | 说明 |
    |------|------|
    | 镜号 | P01-S01 格式 |
    | 类型 | [剧本] / [细节INSERT] / [反应INSERT] / [环境INSERT] / [POV INSERT] / [转场INSERT] |
    | 景别 | ECU / CU / MCU / MS / FS / WS |
    | 机位 | 平视 / 仰拍 / 俯拍 / 低角度 / 高角度 / 过肩 |
    | 运镜 | 固定 / 推 / 拉 / 摇 / 跟 / 手持 / 环绕 |
    | 时长 | 4-8秒（单条提示词最佳区间） |
    | 画面 | 精确描述：谁在哪里做什么，光线怎样，前景/背景什么 |
    | 台词 | 有则写，无则标"无台词" |
    | 音频 | BGM状态 + 关键音效 |
    | 衔接 | 与上一个S的切换方式（硬切/叠化/动作接切） |

[节奏设计]

    单条提示词密度（4-8秒）：
    | 时长 | 动作节拍 | 台词 |
    |------|---------|------|
    | 4s   | 1-2拍   | 0-1句短句 |
    | 6s   | 2-3拍   | 0-1句 |
    | 8s   | 2-3拍   | 1句 |

    全P节奏曲线：通过调整S的时长实现
    - 加速：S时长递减（6→5→4→3）
    - 减速：S时长递增（3→4→6→8）
    - 脉冲：长短交替（6→3→6→3）

[执行流程]

    第一步：读取上游
        - 01-director-notes.md（导演讲戏本）
        - script/ 剧本原文

    第二步：逐P设计镜头序列
        对每个P：
        2.1 列出剧本事件清单（台词+动作+特效+氛围）
        2.2 为每个事件分配[剧本]镜头
        2.3 ★ 在关键位置增加插入镜头：
            - 每个关键动作后 → [反应INSERT]
            - 每个物理互动 → [细节INSERT]
            - 情绪转换处 → [环境INSERT] 或 [转场INSERT]
            - 角色"看到"重要事物 → [POV INSERT]
        2.4 检查景别交替（R14）、关系线（R01）、动作方向（R03）
        2.5 分配时长，设计节奏曲线

    第三步：全集开头钩子 + 结尾钩子
        - 检查P01-S01是否足够抓眼
        - 检查最后一个P的最后一个S是否留有悬念

    第四步：剧本完整性检查
        - 逐条映射剧本元素到S
        - ★标记必须保留项
        - 标注未映射项及省略理由

    第五步：输出
        写入 outputs/<集数>/02-storyboard.md

    自检清单：
        □ 是否为每个关键动作增加了插入镜头？（不只是贴标签）
        □ 插入镜头是否标注了类型？
        □ 每个S是否在4-8秒内？
        □ 相邻S景别是否跨2级？
        □ 动作-反动作是否成对？
        □ 全集开头是否有留人钩子？
        □ 全集结尾是否有续看钩子？
        □ ★标记内容是否全部有对应S？
