# 分镜头脚本示例

> 重点看：[INSERT]标注的镜头——这些是剧本里没有、分镜师基于电影语法增加的。

---

## 示例：航海短剧·��凝雪的决断（P07）

剧本原文只有：洛凝雪做决定→抓手腕→拉去船舱→门前回头说台词→推进船舱→关门→女生反应。

分镜师增加了4个INSERT镜头，让视觉叙事更完整：

### P07-S01 洛凝雪咬唇决断
| 字段 | 内容 |
|------|------|
| 类型 | [剧本] |
| 景别 | MCU |
| 机位 | 平视 |
| 运镜 | 固定 |
| 时长 | 5s |
| 衔接 | 硬切 |

**画面**：灰蓝散射光。洛凝雪面部占画面2/3，眉头缓缓松开，嘴角抿紧，嘴唇颤抖。停顿一秒。咬唇，深吸气。
**台词**：无台词
**音频**：紧张弦乐缓慢渐强，海风底噪

### P07-S02 手抓手腕
| 字段 | 内容 |
|------|------|
| 类型 | [细节INSERT] ← 剧本只写"一把抓住手腕"，分镜增加手部特写 |
| 景别 | ECU |
| 机位 | 平视 |
| 运镜 | 固定 |
| 时长 | 2s |
| 衔接 | 硬切 |

**画面**：灰蓝冷调。纤细的手猛然入画，五指收紧扣住男性手腕，指节发白。被抓的手微微一颤。
**台词**："凌羽，你跟我过来。"（果断，压着情绪）
**音频**：弦乐骤起短音，抓握声清脆

### P07-S03 凌羽微怔
| 字段 | 内容 |
|------|------|
| 类型 | [反应INSERT] ← 动作-反动作原则，剧本没写凌羽此刻的反应 |
| 景别 | CU |
| 机位 | 平视 |
| 运镜 | 固定 |
| 时长 | 2s |
| 衔接 | 硬切 |

**画面**：灰蓝顶光。凌羽面部特写，眼神微怔——不是惊讶，是一瞬间的空白，然后嘴角几不可察地动了一下。
**台词**：无台词
**音频**：弦乐持续，环境音安静

### P07-S04 跟拍背影走向船舱
| 字段 | 内容 |
|------|------|
| 类型 | [剧本] |
| 景别 | FS |
| 机位 | 平视 |
| 运镜 | 手持跟拍 |
| 时长 | 4s |
| 衔接 | 硬切 |

**画��**：洛凝雪拽着凌羽大步走向船舱，长发和粉色裙摆海风中飘动，凌羽踉跄跟着。甲板两侧面色发白的女生目光跟随。
**台词**：无台词
**音频**：弦乐加速，急促脚步声，木板吱嘎

### P07-S05 船舱门前回头
| 字段 | 内容 |
|------|------|
| 类型 | [剧本] |
| 景别 | MS |
| 机位 | 平视 |
| 运镜 | 固定 |
| 时长 | 6s |
| 衔接 | 硬切 |

**画面**：场景切到船舱门前。灰蓝侧光从左上45度照来，在洛凝雪脸上形成明暗分割。她半转身面对甲板方向，脸红如血。身后木门裂纹铁锈，门缝后完全黑暗。
**台词**："都不许靠近船舱！为了大家能活下去……我……我先来！"（声音发颤，声带在抖，但语气坚决）
**音频**：弦乐到达最高点

### P07-S06 门板撞击
| 字段 | 内容 |
|------|------|
| 类型 | [细节INSERT] ← 剧本只写"砰！门关上"，分镜增加门板撞击的视觉冲击 |
| 景别 | ECU |
| 机位 | 平视 |
| 运镜 | 固定 |
| 时长 | 2s |
| 衔接 | 硬切 |

**画面**：门板猛然撞上门框。铁锈铰链震动，盐渍木面弹出一缕灰尘。画面微震。
**台词**：无台词
**音频**：★音画卡点——"砰"的沉重撞击声，同时BGM骤停。之后完全寂静。

### P07-S07 甲板定格·结尾钩子
| 字段 | 内容 |
|------|------|
| 类型 | [反应INSERT] ← 全集结尾钩子，剧本只写"面面相觑"，分镜展开为有续看欲的定格 |
| 景别 | FS |
| 机位 | 平视 |
| 运镜 | 缓慢拉远 |
| 时长 | 5s |
| 衔接 | 硬切 |

**画面**：甲板全景。二十多个女生站着面面相觑。门后隐约传来衣物摩擦声。一个女生不自觉咽了下口水。有人红着脸低头。画面缓缓拉远，定格在沉默中。
**台词**：无台词
**音频**：只有海浪、海风、门后隐约的衣物摩擦声。完全没有音乐。沉默持续到黑屏。

---

### 插入镜头统计

| 镜号 | 类型 | 增值说明 |
|------|------|---------|
| P07-S02 | [细节INSERT] | 手抓手腕的ECU让"抓住"这个动作有了物理质感 |
| P07-S03 | [反应INSERT] | 凌羽的反应让观众感知他的态度（不抗拒） |
| P07-S06 | [细节INSERT] | 门板撞击的ECU配合音画卡点制造全集最强的声画冲击 |
| P07-S07 | [反应INSERT] | 群体反应+衣物摩擦声制造结尾续看欲 |

没有这4个INSERT，P07只是"她抓他走了→说了句话→关门了"。
有了INSERT，P07变成了一个有质感、有反应、有冲击、有余味的完整镜头序列。
