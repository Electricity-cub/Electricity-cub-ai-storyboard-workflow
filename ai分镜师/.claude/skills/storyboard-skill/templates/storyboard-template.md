| 代码 | 景别 | 人体范围 |

| CU | 特写 | 面部 |
| MCU | 中近景 | 胸以上 |
| MS | 中景 | 腰以上 |
| FS | 全景 | 完整人物 |
| WS | 远景 | 人物+环境 |
