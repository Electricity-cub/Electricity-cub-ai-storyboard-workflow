## 写法规范

### 首行（镜头声明）
必须包含：画幅 + 景别 + 机位 + 运镜 + 时长
```
竖屏9:16，极致特写，平视，固定，4秒。
竖屏9:16，全景，低角度仰拍，手持跟拍，5秒。
竖屏9:16，中近景，平视，缓慢推近，6秒。
```

### 主体（画面内容）
- 每句号一个节拍
- 光线作为第一句的修饰
- 动作用精确动词（"五指收紧""猛然伸手""微微一颤"）
- 去掉情绪解释词
- ≤ 150字

### 台词行
- 引号标注
- 括号标注语气
- 无台词标"无台词"

### 音频行
- 必须有
- 格式：音频：[BGM] + [音效]。台词：[语气描述]。
- 标注音画卡点："★砰的瞬间BGM骤停"

## 自检清单

| 检查项 | 要求 |
|--------|------|
| 首行参数 | 景别+机位+运镜+时长全部声明 |
| 字数 | ≤ 200字/条 |
| 情绪词 | 不含"她感到""他觉得""像是" |
| 光线 | 明确写出（一句话） |
| 音频行 | 存在 |
| ★内容 | 分镜标记的必须保留项全部存在 |
| 节拍密度 | 4s≤2拍 / 6s≤3拍 / 8s≤3拍 |
