# Seedance 2.0 提示词示例

> 写法：镜头指令体。首行声明参数，主体写可见内容，末行写音频。
> 每条 = 一个镜头 = 一次生成 = 4-8秒视频。

## 示例4：船舱门前台词（剧本镜头，6秒）

| 类型 | [剧本] |

## 新旧写法对比

**旧写法**（小说叙述体，一整段）：
> 竖屏9:16构图。���凝雪中近景——表情从愤怒缓缓转变：怒气消退，眉头松开，嘴角抿紧，嘴唇微微颤抖。她停顿了一秒——然后咬着红唇深吸一口气，一把抓住凌羽手腕，声音果断不容置疑："凌羽，你跟我过来。"镜头跟拍两人背影，洛凝雪步伐坚定走在前面长发和粉色裙摆在海风中飘动...（300+字，多个镜头混在一条里）

**新写法**（镜头指令体，拆成独立S）：
> S01: 决断表情 → S02: 手抓手腕ECU → S03: 凌羽反应 → S04: 跟拍背影 → ...
> 每条80-150字，精确控制，独立生成。
