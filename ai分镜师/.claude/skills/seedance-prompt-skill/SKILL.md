# Seedance 2.0 视频提示词编写技能

[技能定位——和其他阶段的边界]

    导演讲戏 → 「为什么」（创意指导）
    分镜头脚本 → 「拍什么」（镜头设计）
    ★ 视频提示词 → 「执行」（Seedance可读的镜头指令）

    提示词阶段不做设计决策（分镜已决定），不做情绪分析（讲戏本已完成）。
    提示词阶段的唯一任务：把分镜的每个S翻译成Seedance能理解的精确指令。

[第一性原则]

    1. 中文原生：Seedance对中文理解力最强
    2. 一条提示词 = 一组连续镜头 = 4-15秒视频（段落形式，几个S合一条，带时间标注）
    3. 精确优于丰富：100-250字最佳，超过300字后半段被稀释
    4. 动作优于描述：写"五指收紧扣住手腕"而非"果断地抓住了他"
    5. 参考图已见：@引用的内容不重复描述，只写"变化"

[平台约束]

    | 输入类型 | 上限 |
    |---------|------|
    | 图片 | ≤9张/条 |
    | 视频 | ≤3个/条 |
    | 音频 | ≤3个/条 |
    | 总文件 | ≤12个/条 |
    | 输出 | 4-15秒视频，自带音效配乐 |
    | 限制 | 不支持写实真人脸部素材 |

[★★★ 提示词写法——镜头指令体]

    ★ 核心变革：从「小说叙述体」改为「镜头指令体」。
    不是在写小说的画面描写，而是在给摄影机下达指令。

    结构（三层）：

    第一层：画幅+时长声明（首行）
        格式：画幅，总时长。
        例："竖屏9:16，10秒。"

    第二层：画面段落（主体，连续段落）

        ★★★ 每个镜头必须包含7要素（电影级镜头控制）：

        | 要素 | 说明 | 写法示例 |
        |------|------|---------|
        | ① 焦段 | 决定透视和空间压缩 | 85mm / 50mm / 35mm / 24mm |
        | ② 景别+镜头类型 | 景别+单人/双人/过肩/群体 | 85mm单人特写 / 35mm双人中景 / 50mm过肩中近景 |
        | ③ 机位+运镜 | 角度和摄影机运动 | 平视固定 / 低角度仰拍缓推 / 手持跟拍 |
        | ④ 景深 | 虚化控制 | 浅景深背景虚化 / 深景深全清晰 |
        | ⑤ 前景 | 画面前景元素 | 前景：散乱麻绳虚焦 / 前景：女生肩膀发丝虚焦 |
        | ⑥ 背景 | 画面背景元素 | 背景：灰蓝海天线 / 背景：甲板纵深女生群像 |
        | ⑦ 光线 | 主光方向+色温+明暗 | 灰蓝侧光左上45度，明暗分割 |

        写法规则：
        - 每个镜头用「焦段+景别+镜头类型，机位+运镜，景深」开头声明
          例："85mm单人特写，平视固定，浅景深。前景：甲板木纹虚焦。背景：灰蓝海面散焦。"
        - 镜头切换用「硬切」衔接，切换后立即声明新镜头参数
          例："硬切24mm远景群体镜头，平视固定，深景深。"
        - 光线写在镜头参数之后、动作之前
        - 前景/背景在需要时标注（不是每个镜头都需要，但有纵深层次时必须写）
        - 多个镜头写成一个连贯段落
        - 台词用引号嵌在动作描述中，括号标语气
        - ★ 去掉所有情绪解释词：用动作传达
        - ★ INSERT镜头自然融入段落

        焦段选择指南：
        - 24mm广角：远景/建立镜头，空间纵深感强，适合环境展示
        - 35mm：全景/跟拍，保持环境感的同时聚焦人物
        - 50mm：中景/双人镜头，最接近人眼视角，自然不变形
        - 85mm：特写/单人镜头，空间压缩，背景虚化，突出表情
        - 100-135mm：极致特写，强压缩，极窄景深，适合细节INSERT

    ★★★ 三条补充规则：

    规则1：背景锚定（防穿帮）
        单人镜头/中近景/特写的背景必须根据Master镜头的空间站位来描述。
        例：Master中凌羽站在船头 → Single凌羽的背景应该是"背景：歪斜桅杆和灰蓝海天线虚焦"
        不能写"背景：散焦"了事——要写出从那个角度实际能看到什么。

    规则2：小景别背景中包含其他角色虚焦存在
        拍单人镜头/中近景时，如果场景中有其他角色，
        他们应该在背景中以虚焦方式自然存在（不需要描述他们在做什么反应）。
        这让画面有空间真实感，不会像是把人物P到空白背景上。

    规则3：特写/插入镜头可拼组
        不需要每个插入镜头单独3-5秒出一条。
        多个相关的特写/插入/空镜可以拼成一条10-15秒的提示词，让AI自动处理切镜。
        例：金色粒子落肩膀(5s) + 死寂水汪汪眼睛(6s) → 合为一条11s提示词
        例：门板撞击(3s) + 甲板定格(8s) → 合为一条11s提示词

    第三层：音频指令（末行，必须）
        格式：音频：[BGM] + [音效] + [关键音画卡点]。
        例："音频：紧张弦乐渐强，抓握声清脆，★关门砰的瞬间BGM骤停。"

    ★ 每条提示词末尾统一追加：
    "画面中不要出现任何字幕。"

    ★ 禁止出现：
    - 情绪解释词（"她心动了""他感到震惊""像是在做改变命运的决定"）
    - 文学修辞（"犹如神明降临般""楚楚动人"）——除非是视觉描述需要
    - 长定语从句（"穿着因为躺在甲板上而有些褶皱的白色衬衣的洛凝雪"）
    - 超过200字的单条提示词

[写法对比示例]

    ✗ 旧写法（小说叙述体）：
    "竖屏9:16构图。洛凝雪中近景——表情从愤怒缓缓转变：怒气消退，眉头松开，
    嘴角抿紧，嘴唇微微颤抖。她停顿了一秒——然后咬着红唇深吸一口气，一把抓住
    凌羽手腕，声音果断不容置疑："凌羽，你跟我过来。""

    ✓ 新写法（电影级镜头控制·段落体）：

    "竖屏9:16，12秒。

    85mm单人特写，平视固定，浅景深。前景：甲板粗麻绳虚焦。背景：灰蓝海面散焦光斑。灰蓝侧光从左上45度，在【@图片2】洛凝雪脸上形成明暗分割。面部占画面右侧2/3，眉头缓缓松开，嘴角抿紧，嘴唇颤抖。停顿一秒。咬唇深吸气。硬切120mm极致特写手部，平视固定——纤细的手猛然入画五指收紧扣住男性手腕指节发白，被抓的手微微一颤。"凌羽，你跟我过来。"（果断压抑）硬切85mm单人特写——凌羽面部，眼神微怔一瞬间空白，嘴角几不可察地动了一下。

    音频：紧张弦乐渐强→抓握声清脆有力→弦乐骤起短音。
    画面中不要出现任何字幕。"

    要点：
    - 每个镜头声明焦段+景别+镜头类型+机位+运镜+景深（7要素）
    - 前景/背景在有纵深层次时明确标注
    - 光线精确到方向角度和明暗关系
    - 动作用精确物理描述（"五指收紧""指节发白""微微一颤"）
    - 段落形式，多镜头一组，硬切衔接
    - 动作精确（"眉头松开→嘴角抿紧→嘴唇颤抖→停顿→咬唇→伸手"）
    - 台词独立行+语气标注
    - 没有"像是在做改变命运的决定"这类情绪解释
    - 音频独立行

[@ 引用语法]

    | 用途 | 写法 |
    |------|----- |
    | 人物形象 | 【@图片1】角色名 |
    | 场景背景 | 场景参考【@图片3】 |
    | 首帧 | 【@图片1】作为首帧 |

    引用放在人物/场景首次出现处。同一条提示词内不重复引用。

[节拍密度约束]

    1拍 ≈ 2.5秒。

    | 时长 | 最大节拍 | 最大台词 |
    |------|---------|---------|
    | 4s   | 1-2拍   | 1句短句 |
    | 6s   | 2-3拍   | 1句 |
    | 8s   | 3拍     | 1-2句短句 |

    头尾安全区：前0.5s和后0.5s不放关键内容。

[执行流程]

    第一步：读取上游
        - 02-storyboard.md（分镜头脚本）
        - 01-director-notes.md（导演讲戏本）
        - assets/character-prompts.md + scene-prompts.md
        - examples/prompt-examples.md

    第二步：建立素材对应表
        先人物后场景，分配@引用编号

    第三步：编写段落式提示词
        将分镜的多个S合为一条提示词（段落形式），带时间标注：
        3.1 每个镜头带【Xs-Xs】时间标注
        3.2 每个镜头声明7要素（焦段+景别·镜头类型+机位+运镜+景深+前景+背景+光线）
        3.3 镜头之间用"硬切"衔接
        3.4 INSERT镜头自然融入段落
        3.5 检查字数 ≤ 300字/条

    第四步：输出
        写入 outputs/<集数>/03-seedance-prompts.md

    自检清单：
        □ 每个镜头是否带时间标注【Xs-Xs】？
        □ 焦段+景别+镜头类型+机位+运镜+景深+前景+背景+光线 是否声明？
        □ 光线是否明确写出？
        □ 音频行是否存在？
        □ 是否末尾加"画面中不要出现任何字幕。"？
        □ ★标记内容是否全部保留？
        □ 去掉情绪解释词了吗？

[群演处理]
    - 不分配@引用
    - 直接用外观描述："长发披肩面色苍白的女生""扎马尾穿JK制服的女生"
    - 描述要足够具体让模型区分不同人

[禁忌规则]

    绝对禁忌：
    - 否定句（"不要移动" → "保持静止"）
    - 情绪解释（"她心动了" → 用停顿/颤抖/咬唇传达）
    - 文学修辞堆砌（"犹如""仿佛""宛若"——除非描述视觉效果需要）
    - 单条超200字
    - 首行不声明镜头参数
    - 重复参考图已有的静态内容
    - @引用不指定用途
