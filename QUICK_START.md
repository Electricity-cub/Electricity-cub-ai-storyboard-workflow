# 快速开始指南

## 🚀 5分钟快速上手

### 步骤1：启动后端服务

```bash
# 进入后端目录
cd backend

# 启动服务
python3 app.py
```

看到以下输出表示启动成功：
```
AI分镜师后端服务启动中...
Coze API URL: https://3b7j5mjhsz.coze.site/run
Token已配置: 是
 * Running on http://127.0.0.1:5000
```

### 步骤2：打开前端页面

**推荐使用异步版本（无超时问题）：**

在浏览器中打开 `index_async.html` 文件

**或使用同步版本（兼容但可能超时）：**

在浏览器中打开 `index_new.html` 文件

### 步骤3：输入剧本内容

示例剧本（建议< 300字）：
```
小明和小红在公园相遇，一见钟情。他们开始聊天，
发现彼此有很多共同点，约定下次再见。
```

### 步骤4：选择参数

- **集数**：ep01
- **视觉风格**：写实/动漫/国风/科幻/水墨
- **项目类型**：国内短剧/电影/动画/广告

### 步骤5：生成

点击"生成分镜"按钮，等待结果

## 🔍 常见问题快速解决

### 问题1：连接失败

**症状：** "Failed to fetch"

**解决：**
```bash
# 检查后端是否运行
curl http://localhost:5000/health

# 如果没有运行，启动后端
cd backend && python3 app.py
```

### 问题2：任务失败

**症状：** 任务状态为 "failed"

**解决：**
1. 查看错误信息
2. 简化剧本内容
3. 稍后重试

### 问题3：超时

**症状：** "504 GATEWAY TIMEOUT" 或 "任务超时"

**解决：**
1. 使用异步版本（index_async.html）
2. 简化剧本内容（< 300字）
3. 减少场景数量

## 📱 API接口测试

### 创建异步任务

```bash
curl -X POST http://localhost:5000/api/v1/tasks \
  -H "Content-Type: application/json" \
  -d '{
    "input": {
      "script_content": "测试剧本",
      "episode_number": "ep01",
      "visual_style": "写实",
      "project_type": "国内短剧"
    }
  }'
```

### 查询任务状态

```bash
# 替换 {task_id} 为实际的任务ID
curl http://localhost:5000/api/v1/tasks/{task_id}
```

### 健康检查

```bash
curl http://localhost:5000/health
```
